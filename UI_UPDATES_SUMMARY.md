## 🔄 **資料流程**

### 心情溫度計投票流程
1. 用戶選擇心情區間
2. 選擇投票成員
3. 系統自動創建或找到用戶
4. 寫入心情記錄到資料庫
5. 更新本地顯示
6. 保持 localStorage 備份

### 簽到樹簽到流程
1. 用戶點擊樹上位置
2. 選擇簽到成員
3. 系統自動創建或找到用戶
4. 寫入簽到記錄到資料庫
5. 在樹上顯示簽到印章
6. 保持 localStorage 備份

## 📊 **資料庫結構**

### 用戶表 (User)
- 自動創建投票/簽到用戶
- 包含基本資訊和角色

### 心情記錄表 (Mood)
- 記錄心情數值 (1-5)
- 關聯到用戶
- 包含備註和時間

### 簽到記錄表 (Checkin)
- 記錄簽到時間
- 關聯到用戶
- 支援課程關聯
