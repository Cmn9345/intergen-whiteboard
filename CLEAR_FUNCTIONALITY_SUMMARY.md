## 🔄 **清除流程**

### 心情紀錄清除流程
1. 用戶在管理儀表板點擊「查看今日心情紀錄」
2. 進入心情紀錄頁面
3. 點擊右下角紅色刪除按鈕
4. 確認清除對話框：「您確定要清除所有今日心情記錄嗎？」
5. 選擇「確認清除」
6. 系統執行：
   - 刪除資料庫中的心情記錄
   - 清除 localStorage 中的投票數據
   - 發送 `CLEAR_MOOD_DATA` 指令
   - 心情溫度計頁面接收指令並清除畫面
7. 顯示成功訊息：「今日心情記錄已清除，心情溫度計畫面也已重置」

### 簽到紀錄清除流程
1. 用戶在管理儀表板點擊「查看今日簽到紀錄」
2. 進入簽到紀錄頁面
3. 點擊右下角紅色刪除按鈕
4. 確認清除對話框：「您確定要清除所有今日簽到記錄嗎？」
5. 選擇「確認清除」
6. 系統執行：
   - 刪除資料庫中的簽到記錄
   - 清除 localStorage 中的簽到數據
   - 發送 `CLEAR_CHECKIN_DATA` 指令
   - 簽到樹頁面接收指令並清除畫面
7. 顯示成功訊息：「今日簽到記錄已清除，簽到樹畫面也已重置」

## 🎯 **技術實現**

### 跨頁面通信
```javascript
// 發送清除指令
window.postMessage({
  type: 'CLEAR_MOOD_DATA', // 或 'CLEAR_CHECKIN_DATA'
  timestamp: Date.now()
}, '*');

// 接收清除指令
window.addEventListener('message', (event) => {
  if (event.data.type === 'CLEAR_MOOD_DATA') {
    // 清除心情溫度計數據
  }
  if (event.data.type === 'CLEAR_CHECKIN_DATA') {
    // 清除簽到樹數據
  }
});
```

### 清除的數據
#### 心情溫度計
- `moodThermometerVotes:v4` - 投票數據

#### 簽到樹

- `checkinTreeStamps:startTime` - 開始時間
